# Rust 引用、借用与切片笔记

## 1. 引用（References）

### 不可变引用

```rust
let value = 42;            // i32
let ref_of_value = &value; // &i32 ，不可变引用
```

- 默认创建的引用是**不可变的**。
- 引用允许你访问数据但不会获取所有权。

### 解引用

```rust
println!("Value is {}", *ref_of_value);  // 手动解引用
println!("Value is {}", ref_of_value);   // 自动解引用（在某些上下文中）
```

- `*ref_of_value` 手动获取引用指向的值。
- `println!` 等宏可以自动解引用，所以两种写法都能工作。

### 查看地址

```rust
println!("value's addr is {:p}", ref_of_value);          // 被引用者的地址
println!("ref_of_value's addr is {:p}", &ref_of_value);  // 引用变量本身的地址
```

- `{:p}` 格式化占位符用于打印指针/地址。
- `ref_of_value` 存的就是 `value` 的地址，所以直接打印显示的是 `value` 的地址。
- `&ref_of_value` 是获取引用变量自身在栈上的地址。

## 2. 借用（Borrowing）

### 不可变借用

```rust
let mut num1 = 42;
let ref_num1 = &num1;   // 不可变借用，此时不能通过 ref_num1 修改 num1
// ref_num1 只是只读访问，num1 本身可变，但在借用存在期间不能修改
```

- 不可变借用允许同时存在**多个**。
- 在不可变借用存在期间，原所有者即使声明为 `mut` 也不能修改数据（除非借用已经用完）。

### 可变借用

```rust
let mut num1 = 42;
{
    let ref_num1 = &mut num1;   // 可变借用
    *ref_num1 += 1;             // 通过可变引用修改值
} // 借用结束，num1 可再次使用
```

- 同一个作用域内，对同一个数据**只能存在一个可变借用**。
- 可变借用与不可变借用**不能同时共存**。

### 借用规则示例（正确写法）

```rust
let mut v1 = 1;
{
    let v2 = &v1;               // 不可变借用
    println!("v2: {}", v2);     // 使用不可变借用
} // v2 离开作用域，不可变借用结束

let v3: &mut i32 = &mut v1;     // 此时可以创建可变借用
*v3 = 10;
println!("v1: {}", v1);
```

- 通过限制借用作用域，可以依次使用不可变借用和可变借用。
- Rust 的借用检查器在编译时确保内存安全，避免了数据竞争。

## 3. 切片（Slice）

切片是一种**借用**，它引用集合中的一段连续元素，分为不可变切片和可变切片。

```rust
let arr: [i32; 5] = [1, 2, 3, 4, 5];

// 不可变切片
let s1 = &arr[1..3];    // 索引 [1, 3) ，包含元素 2 和 3
let _all = &arr[..];    // 全部元素

println!("{:?}", s1);   // 输出 [2, 3]

// 可变切片
let mut arr = [1, 2, 3, 4, 5];
let s2 = &mut arr[1..4]; // 可变切片
s2[1] = 10;              // 修改原数组的第 2 个元素（索引 2）
println!("{:?}", arr);   // [1, 2, 10, 4, 5]
```

- `[start..end]` 语法表示左闭右开区间。
- `[start..=end]` 表示包含 `end` 的闭区间。
- 切片保存了原始数据的指针和长度，因此操作切片会直接影响原数据。

## 4. 函数与借用

```rust
fn print_value(arg: &i32) {
    println!("{}", arg);
}

fn main() {
    let x = 5;
    print_value(&x);   // 传递不可变引用
}
```

- 函数参数用 `&T` 表示不可变借用，调用时直接传入引用。
- 函数不会获取所有权，只是“借用”数据来读取。

---

## 小结

- **引用** 是获取数据的一种方式，分为不可变引用 `&T` 和可变引用 `&mut T`。
- **借用规则**：在任意时刻，要么只有一个可变引用，要么任意数量的不可变引用，两者不能并存。
- **切片** 是引用的一种特殊形式，用于引用集合的一部分。
- 牢记借用规则，能帮助你在编译期就避免内存问题，写出安全的 Rust 代码。
